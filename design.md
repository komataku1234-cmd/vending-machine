# 要件定義

扱う商品：水：100、お茶：150、コーラ：200
扱う紙幣：10、50、100、500
売り切れや購入はコンソールに出力される（画面上では反映してない）
キャンセルは買ったものを配列に入れることで再現した。
実際に使用する場合（財布からお金を入れる）を想像し作成した。

# 設計

ボタンでの飲み物購入（ログで出力）
お金をドラックして指定の位置のドロップすることで金額を増やす。
在庫の有無で購入できるか判断
在庫数：
水：3
お茶：2
コーラ：1

# 関数一覧
buy(name)	飲み物購入処理
cancel1()	最後に購入した商品を返金
createCoinSpawner(id, image, value)	元コインをドラッグした時に新しいコインを生成する設定
createCoin(image, value)	コイン要素を生成する
moveCoin(coin, event)	コインをマウス位置へ移動する
makeDraggable(coin)	生成済みコインを再度ドラッグできるようにする
checkHit(coin)	投入口に入ったか判定する
updateMoney()	金額表示を更新する
