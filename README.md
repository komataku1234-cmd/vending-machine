# 🥤 自動販売機アプリ

JavaScript・HTML・CSSを使って作成した自動販売機シミュレーターです。

## 概要

コインを投入し、商品ボタンを押して飲み物を購入できるWebアプリです。

### 購入可能な商品

| 商品 | 価格 |
|--------|--------|
| 水 | 100円 |
| お茶 | 150円 |
| コーラ | 200円 |

## 使用技術

- HTML
- CSS
- JavaScript

## 機能

- 10円・50円・100円・500円硬貨の投入
- 所持金のリアルタイム表示
- 商品購入機能
- 購入時の残高計算
- 商品取り出し口への表示
- レスポンシブ対応

## 起動方法

1. リポジトリをクローン

```bash
git clone git@github.com:komataku1234-cmd/vending-machine.git
```

2. プロジェクトへ移動

```bash
cd vending-machine
```

3. `index.html` をブラウザで開く
